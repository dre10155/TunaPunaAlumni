Code of conduct

The Code does not supersede other policies or agreements the Association has in place. This Code of conduct serves as a basis for ethical behaviour and decision-making in the conduct of the of the Association's code of conduct and where appropriate adhere to that code

The objectives of the Code are to:

  • Provide direction to members and those working with the Association to uphold its values and standards

  • Promote professionalism and excellence;

  • Express shared assumptions and organisational values;

  • Adhere to the guidelines of the Associations' social responsibilities as outlined in the constitution.

Members are strongly encouraged to follow the reporting guidelines set up by this Code. Failure to act in accordance with the Code of Conduct may result in investigation under the relevant provisions of the Alumni Constitution. This can result in disciplinary action including expulsion.
